Electronic library

A virtual collection of resources that are accessible electronically 

     
It provides users with the ability to access and retrieve information, documents, books, journals, 
and other digital materials using computers or mobile devices connected to the internet.

Electronic libraries offer numerous advantages over traditional physical libraries: 
- provide convenient and instant access to a vast amount of information from anywhere in the world, 
eliminating the need to visit a physical library. 
- Users can search for specific keywords or topics to find relevant materials quickly. 
- that are not limited by physical space constraints.
- they can store a wide range of digital formats, including text documents, images, audio files, and videos.

HTML , CSS , JavaScript at Software Engineering 

 
HTML (Hypertext Markup Language) is a markup language for web pages, 
uses HTML to arrange and organize the content of a web page and determine its structure 

CSS (Cascading Style Sheets) is a formatting language used to change the appearance and 
visual formatting of web pages 
 
JS (JavaScript ) is a powerful and versatile programming language widely used in web development 
to perform many different tasks 

The three work together to achieve an attractive design and advanced functionality on web pages
